# NovaDev Code Studio site

This folder contains a ready-to-publish static homepage for GitHub Pages.

Files:
- index.html
- assets/novadev-logo.png
- assets/matchscore-pro-banner.png
- assets/eternal-light-banner.png
- assets/novadev-hero-preview.png
- assets/favicon.png

Before publishing:
1. Replace the sample email address:
   hello@novadevcodestudio.com

2. Project links are already wired:
   - MatchScore Pro: https://play.google.com/store/apps/details?id=space.manus.matchscore.t20260326123000
   - Eternal Light: https://play.google.com/store/apps/details?id=com.eternallight.christianprayer
     Note: Eternal Light is still in closed testing.

3. If you want, update the GitHub profile link:
   https://github.com/yurybreiapps-jpg

Publish on GitHub Pages:
1. Upload these files to your repository folder.
2. Commit and push.
3. In GitHub Pages, point the site to the branch/folder you want to publish.

Notes:
- The site is fully static and GitHub Pages friendly.
- The hero uses the selected single-N logo and stronger colored light lines.
- You can add more project cards by duplicating one .project-card block in index.html.
